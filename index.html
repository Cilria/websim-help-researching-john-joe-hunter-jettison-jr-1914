Rubidium (web browser)

Rubidium is a discontinued web browser developed by Rubidium Software S.A., a French software company. Released in 1999, it was marketed as "the fastest and most powerful web browser" designed specifically for Windows 98. Rubidium was positioned as a "Pro Browser" targeting power users, developers, and enterprise customers seeking superior performance and advanced features.

History

Rubidium Software S.A. was founded in 1999 in Lyon, France by former Netscape engineers Jean-Pierre Dubois and Marie-Claire Laurent. The company was established with the goal of creating a web browser that would outperform existing solutions on Windows 98, which was the dominant consumer operating system at the time.

Development

The browser was developed over 18 months using a custom rendering engine called RubidiumHTML, optimized for the limitations of Windows 98. The name "Rubidium" was chosen for its association with speed (rubidium is the second-fastest diffusing alkali metal) and its French origin.

Release timeline

  • October 15, 1999: Rubidium 1.0 released
  • March 12, 2000: Rubidium 2.0 adds JavaScript 1.5 support
  • September 8, 2000: Rubidium 3.0 introduces tabbed browsing
  • January 20, 2001: Rubidium 3.5 switches to Chromium engine
  • October 12, 2002: Final version 3.7.2 released

Features

Rubidium was known for its advanced features that were ahead of its time:

Tabbed browsing

Rubidium introduced tabbed browsing to Windows users before Internet Explorer, allowing users to open multiple web pages in a single window.

Built-in developer tools

The browser included an integrated debugger, DOM inspector, and network monitor, making it popular among web developers.

Advanced rendering engine

RubidiumHTML featured early support for CSS2 and partial CSS3, along with advanced JavaScript optimization that provided superior performance on Windows 98 systems.

Security features

  • Built-in popup blocker
  • Anti-phishing protection
  • Malware scanning integration
  • Encrypted bookmark synchronization

Performance

Rubidium was renowned for its speed optimizations:

Memory management

The browser used a custom memory allocator that reduced Windows 98 system crashes by 40% compared to Internet Explorer 5.0.

Rendering speed

Independent benchmarks showed Rubidium loading complex web pages 2.3x faster than Netscape Communicator and 1.8x faster than Internet Explorer on identical hardware.

Market reception

Rubidium gained a dedicated following among power users and developers, reaching approximately 2.3 million active users at its peak in 2001. However, it struggled to gain mainstream adoption due to:

  • Limited marketing budget compared to Microsoft
  • Compatibility issues with some ActiveX-based websites
  • Lack of corporate IT department support

Discontinuation

On December 15, 2002, Rubidium Software S.A. announced the discontinuation of Rubidium browser, citing:

  • Financial difficulties following the dot-com crash
  • Inability to compete with Internet Explorer's market dominance
  • Rising development costs for Windows XP compatibility

The company transitioned to providing enterprise web development tools before being acquired by SAP in 2005.

Legacy

Despite its relatively short lifespan, Rubidium influenced several modern browser features:

  • Tabbed browsing became standard in all major browsers
  • Built-in developer tools are now universal
  • Performance optimization techniques were adopted by Firefox and Chrome

The RubidiumHTML rendering engine was open-sourced in 2003 and contributed to the development of early WebKit components.
